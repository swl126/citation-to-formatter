# APA 7th Edition

Source: [Pitt Library — APA 7th Edition](https://pitt.libguides.com/citationhelp/apa7)

## In-text form
Parenthetical: `(Author, Year)`, e.g. `(Sapolsky, 2017)`. Include a page number after a comma for direct quotes: `(Sapolsky, 2017, p. 45)`.
- 2 authors: `(Smith & Jones, 2020)` — always `&` inside parentheses, "and" if the names are in running text.
- 3+ authors: first author + `et al.` — `(Grady et al., 2019)`.
- No page numbers available (websites, some ebooks): use a paragraph number, heading, or section name instead if quoting directly; otherwise the year alone is sufficient for a paraphrase.

## Author formatting (reference list)
`Last, F. M.` — last name first, initials only for first/middle names, for every author. Join multiple authors with commas and `&` before the last: `Grady, J. S., Her, M., Moreno, G., Perez, C., & Yelinek, J.`

Notes:
- Use the same format for print books and ebooks — for ebooks, the device/format (e.g., Kindle) is **not** included in the reference.
- Journal article titles: sentence case, no quotation marks, not italicized. Journal name + volume: italicized; issue number in parentheses, not italicized.
- Most of these examples originate from APA's own Reference Examples page.

## Missing information
- **No author**: move the title to the author position, e.g. *Title of work*. (Year). Publisher.
- **No date**: use `(n.d.)` in place of the year, both in-text and in the reference.
- **No page numbers** (e.g. websites, some ebooks): for a direct quote, cite a paragraph number, heading, or section name instead — e.g. `(Author, Year, Introduction section)`.

## Generative AI in APA
Treat the AI company as author/publisher, and the specific model as the "Large Language Model" or tool. Cite a specific generated chat as `[Generative AI chat]` with a shareable link if available; cite the tool generally as `[Large Language Model]` with the tool's homepage.

## Multiple-authors quick check
- 1 author: `Last, F. M.`
- 2 authors: `Last, F. M., & Last, F. M.`
- 3–20 authors: list all, `&` before the last.
- 21+ authors: list first 19, ellipsis, then the final author.
- In-text: 1 author → full name every time; 2 authors → both names every time (`&`); 3+ authors → first author + `et al.` from the first citation onward.
