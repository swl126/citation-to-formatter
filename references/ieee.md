# IEEE Style

Source: [Pitt Library — IEEE Style](https://pitt.libguides.com/citationhelp/ieee)

Used by the Institute for Electrical and Electronics Engineers — engineering, computer science, information technology.

## In-text citing
- No author name, page, or date in-text — only a **bracketed number**, e.g. `[1]`, referring to the numbered reference list entry.
- Number sources **in the order first cited** (not alphabetically) — once a source has a number, reuse that same number every time it's cited again.
- Place the bracketed number within the line of text, before punctuation, with a space before the first bracket.
- Multiple sources at once: list each number in its own brackets, separated by commas or a dash: `[1], [3], [5]` or `[1]–[5]`.

Examples (verbatim from Pitt guide):
- "...end of the line for my research [13]."
- "This theory was first put forward in 1987 [1]."
- "Scholtz [2] has argued that..."
- "Several recent studies [3], [4], [15], [16] have suggested that..."
- "For example, see [7]."

## Reference list
- Title it **References**, centered or left-aligned at the top.
- List entries **numerically in citation order** (not alphabetical), bracketed number flush left, hanging indent for wrapped lines.
- Author name format: `F. Last` (first initial, then last name) for **every** author — never inverted, even for the first author. Example: Adel Al Muhairy → `A. Al Muhairy` (not `Al Muhairy, Adel`).
- Article/chapter titles: in quotation marks.
- Journal/book titles: italicized.
